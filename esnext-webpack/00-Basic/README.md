# Aurelia-Auth0 
This application is Aurelia version of the 00-Basic application


### Running The App

```
npm install
```

or 

```
yarn install
```

followed by 
```
nps
```


**Note**: the command **[`nps`](https://www.npmjs.com/package/nps)** is a better alternative to **`au run --watch`** when using webpack as the compiler / bundler.
